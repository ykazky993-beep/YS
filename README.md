# YS
Just YourShell

# About
YourShell its just a mini project custom shell using Python using `subprocess`

# Using
 Just
 `git clone https://github.com/ykazky-993/YS.git`
 `cd YS/yourshell`
 `python3 yourshell.py`

 in YourShell
 `fdel pkg update`
 `fdel pkg upgrade`

 or for file management
 `fdel -h`

 and YourShell have special feature
 `ysGhost`
 `ysGhost-x` PC only!
 `ysGhost-m` For termux or PC
 `ysGhost-v` For termux & PC but some packages is not found
 `ysGhost-v-install` for install packages you can configure it at `lock/install_all.sh`
